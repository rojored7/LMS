# MÓDULO 06: APIs DE SEGURIDAD (JWS/JWE/JOSE)

**Duración**: 4 horas (1.5h teoría + 2h práctica + 0.5h evaluación)
**Nivel**: Intermedio
**Prerequisitos**: Módulos 03-05 completados

---

## 🎯 Objetivos de Aprendizaje

- [ ] Implementar JWS (JSON Web Signature)
- [ ] Implementar JWE (JSON Web Encryption)
- [ ] Trabajar con JWT
- [ ] Streaming seguro con detached signatures
- [ ] Integrar ANKASecure en APIs REST

---

## 📚 Contenido Teórico (1.5 horas)

### 1. Introducción a JOSE (30 min)
- JWS, JWE, JWK, JWA, JWT

### 2. JWS - Firmas (25 min)
- Estructura: Header, Payload, Signature
- Algoritmos: HS256, RS256, ES256

### 3. JWE - Cifrado (25 min)
- Content encryption + Key encryption
- A256GCM, RSA-OAEP

### 4. JWT y Streaming (20 min)
- Claims estándar
- Detached-JWS para archivos grandes
