# MÓDULO 07: NORMATIVAS Y CUMPLIMIENTO

**Duración**: 3 horas (1.5h teoría + 1h casos prácticos + 0.5h evaluación)
**Nivel**: Intermedio
**Prerequisitos**: Conocimientos de seguridad general

---

## 🎯 Objetivos de Aprendizaje

- [ ] Explicar requisitos de cumplimiento principales
- [ ] Aplicar controles según normativas
- [ ] Realizar auditorías de cumplimiento
- [ ] Documentar evidencias
- [ ] Alinear soluciones técnicas con regulaciones

---

## 📚 Contenido Teórico (1.5 horas)

### 1. NIST (20 min)
- NIST Cybersecurity Framework
- NIST PQC Project

### 2. FIPS 140-2/140-3 (15 min)
- Niveles de seguridad (1-4)
- Módulos criptográficos validados

### 3. PCI DSS (15 min)
- 12 requisitos
- Protección de datos de tarjetas

### 4. HIPAA (15 min)
- Protección de PHI
- Requisitos técnicos

### 5. GDPR (15 min)
- Protección de datos personales
- Derecho al olvido

### 6. NSA CNSA 2.0 (15 min)
- Transición a PQC
- Fechas límite

### Caso 01: Auditoría PCI DSS (20 min)
### Caso 02: Migración PQC según CNSA (20 min)
### Caso 03: Cumplimiento HIPAA (20 min)
