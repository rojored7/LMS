# Bibliografía - Seguridad en Sistemas Operativos

## Libros Fundamentales

1. **"SELinux by Example" - Frank Mayer, David Caplan, Karl MacMillan**
   - ISBN: 978-0131963702
   - Guía práctica de SELinux con casos de uso reales
   - Nivel: Intermedio

2. **"SSH Mastery" - Michael W. Lucas**
   - ISBN: 978-1642350029
   - Configuración avanzada de OpenSSH
   - Nivel: Intermedio-Avanzado

3. **"Linux Security Cookbook" - Daniel J. Barrett, Richard E. Silverman**
   - ISBN: 978-0596003913
   - Recetas prácticas de seguridad Linux
   - Nivel: Todos los niveles

4. **"Practical Linux Security" - Tajinder Kalsi**
   - ISBN: 978-1786463555
   - Hardening completo de sistemas Linux
   - Nivel: Intermedio

5. **"Linux Kernel Development" - Robert Love**
   - ISBN: 978-0672329463
   - Para entender seguridad a nivel kernel
   - Nivel: Avanzado
