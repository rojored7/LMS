# MÓDULO 05: GESTIÓN DE CLAVES Y PKI

**Duración**: 4 horas (1.5h teoría + 2h práctica + 0.5h evaluación)
**Nivel**: Intermedio-Avanzado
**Prerequisitos**: Módulos 03-04 completados

---

## 🎯 Objetivos de Aprendizaje

- [ ] Diseñar sistemas de gestión de claves robustos
- [ ] Implementar rotación y revocación de claves
- [ ] Trabajar con HSMs
- [ ] Administrar PKI completa
- [ ] Migrar claves PKCS#12

---

## 📚 Contenido Teórico (1.5 horas)

### 1. Ciclo de Vida de Claves (30 min)
- Generación, distribución, almacenamiento
- Uso, rotación, revocación, destrucción

### 2. Almacenamiento Seguro (20 min)
- HSM, TPM
- Key wrapping
- AWS KMS, Azure Key Vault

### 3. PKI (30 min)
- Root CA, Intermediate CA
- Certificados X.509
- CRL, OCSP

### 4. Gestión en ANKASecure (10 min)
